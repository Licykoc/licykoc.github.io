分割绳子

其实这道题是真的水，就是小数二分然而我调了好久
暴力的话应该可以那个50pts?我没试过，但是应该很好想把

正解的二分其实也很水，我们先看看普通二分的模板:

l=1; r=max;
 
 while l<=r do begin
   mid=(l+r) div 2;
   if check(mid) then l:=mid+1 else r:=mid-1;
 end;

write(...);

很简单吧，那换成小数怎么做呢？
其实也很简单
就是这一行 while l<=r do改成while (r-l)>小数精度 do
就ok了

题意很简单，n个木棍切成k个小段，求每个小段最长长度
不说别的了，上代码

uses math;
var
  mid,l,r:extended;
  n,k,i:longint;
  a:array[0..10000] of extended;
function check (x:extended):boolean; //判断函数
  var i:longint;
  begin
    s:=0;
	for i:=1 to n do s:=s+trunc(a[i]/x);
	if s>=k then exit(true) else exit(false);
	// s>=k 想想看是什么意思
  end;
begin
  read(n,k);
  for i:=1 to n do begin read(a[i]); r:=max(r,a[i]); end; 
  //取r的范围
  l:=1; r:=trunc(r)+1;
  while (r-l)>0.001 do begin 
  //二分模板
    mid:=(l+r)/2;
	if check(mid) then l:=mid+0.0001 else r:=mid-0.0001;

  end;
  write(l:0:2); //输出
end.
